# ДАННЫЕ

### челики
* id (PK)
* паспорт
* инн (пин)
* имя фамилия

### логины
* id челика (FK) -> челики(id)
* username (PK)
* хэш пароля

### роли
* id (PK)
* название

### роли челиков
* id челика (FK) -> челики(id)
* id роли (FK) -> роли(id)

### задачи
* id (PK)
* кто поставил (FK) -> челики(id)
* когда поставил
* суть задачи / описание

### назначения
* кто назначил (FK) -> челики(id)
* на кого назначил (FK) -> челики(id)
* когда
* какую задачу (FK) -> задачи(id)

### выполнение - кто когда сколько что делал
* кто (FK) -> челики(id)
* время начала
* время окончания
* задача (FK) -> задачи(id)
